# Ayush-Garden
 AYUSH Garden: An Interactive Virtual Plant Library

Welcome to AYUSH Garden, an interactive web application that allows you to explore the world of medicinal plants used in traditional AYUSH systems (Ayurveda, Yoga & Naturopathy, Unani, Siddha, and Homeopathy).

## Features

*   **Virtual Garden:** Browse a curated collection of medicinal plants, each with detailed information.
*   **Explorer Map:** An interactive map to visually discover the location of different plants in the virtual garden.
*   **AI Plant Identifier:** Use your device's camera to identify a plant and get AYUSH-related information about it.
*   **Personalized Recommendations:** Get AI-powered suggestions for plants based on your wellness goals.
*   **Plant Knowledge Quiz:** Test your knowledge about the medicinal plants in the garden.
*   **Community Contributions:** Share your own knowledge by submitting plants to the community-sourced library.

## Tech Stack

*   **Framework:** Next.js (App Router)
*   **UI:** React, TypeScript, ShadCN UI, Tailwind CSS
*   **Generative AI:** Google's Gemini models via Genkit
*   **Database & Auth:** Firebase (Firestore & Authentication)
